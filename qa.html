老李说Web

路漫漫其修远兮，吾将上下而求索。

Web 是什么？

所有联网的电脑，智能设备，互联互通，组成的一个巨大的网络系统。

为什么学习 Web？

Web 从1995年诞生至今，已经形成了成熟的产业，学习它可能是一种技能，一种职业，或者一种事业。

怎么学习 Web？

Web 是一个复杂的系统，分而治之的思想，基本分为前端和后端。

前端三大基础：HTML/CSS/JavaScript。衍生出诸多框架、工具。

最好的教程是官方教程，个别官方教程有点啰嗦，或晦涩，可以借助些书籍，一步一步，由浅入深的坚持学习。
