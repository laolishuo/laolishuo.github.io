老李说Web

一步一个脚印，即使很轻，也不要停下。

HTML

HTML 超文本标记语言，用来描述网页的结构和内容。

CSS

CSS 层叠样式表，用来描述网页的布局和样式。

JavaScript

JavaScript 是一种用于为网站增加交互特性的编程语言。

Node.js

Node.js 是一个开源、跨平台的 JavaScript 运行时环境。

Vue.js

Vue 是一款用于构建用户界面的 JavaScript 框架。它基于标准 HTML、CSS 和 JavaScript 构建，并提供了一套声明式的、组件化的编程模型，帮助你高效地开发用户界面。

uni-app

uni-app 是一个使用 Vue.js 开发所有前端应用的框架，开发者编写一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序、快应用等多个平台。

TypeScript

TypeScript 是 JavaScript 的超集，它可以编译成纯 JavaScript 。

React

React 用于构建 Web 和原生交互界面的库。
